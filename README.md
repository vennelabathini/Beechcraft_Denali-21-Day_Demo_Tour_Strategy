# Beechcraft_Denali-21-Day_Demo_Tour_Strategy
A competition-grade Tableau strategy dashboard built for the OSU Datathon 2023, designing a data-driven go-to-market plan to launch the Beechcraft Denali turboprop. It models a 40-stop, 21-day demo tour optimized to reach the highest-value competitor-owner prospects, with a full cost-and-ROI case.
